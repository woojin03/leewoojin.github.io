## ✅ Git 푸시 명령어

```bash
git add .
git commit -m "<타입>: <작업 내용 요약>"
git push origin main
```

| 타입      | 의미                             |
|-----------|------------------------------|
| feat      | 새로운 기능 추가                  |
| fix       | 버그 수정                         |
| docs      | 문서 수정 (README 등)             |
| style     | 코드 포맷팅, 세미콜론 누락 등     |
| refactor  | 코드 리팩토링 (기능 변화 없음)     |
| test      | 테스트 코드 추가                  |
| chore     | 빌드/패키지 설정 등 기타 변경     |
